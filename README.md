# 🫶 Hearthands

Care Work for coding agents.

Hearthands is a grounded alternative to power-fantasy agent methodologies. It keeps the useful habits: clarify intent, test proportionate to risk, debug from evidence, review with rigor, and finish work cleanly. It rejects contempt, heroics, and process theater.

Install this repo as a skill/plugin package in Claude, Codex, Gemini/Agy, OpenCode, or any host that reads `skills/`.
