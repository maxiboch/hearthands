# Hearthands Goal

Hearthands is a care-oriented replacement lineage for Superpowers: a skill ecosystem for coding agents that preserves practical workflow discipline while reframing each behavior through maintenance, evidence, sustainability, historical grounding, and respect for the people who live with the work.

The project treats Superpowers as useful upstream inventory, not as the final frame. Hearthands identifies which agent skills are worth carrying forward, studies the values and working style in Max's local agent-history corpus, and rewrites each accepted skill from care-based first principles rather than cosmetic renaming.

The replacement work is guided by these commitments:

- Evidence before confidence.
- Maintenance before heroics.
- Reversibility before spectacle.
- Smallest responsible ownership.
- Historical grounding before ahistorical productivity theater.
- Taste, identity, and lived context as real technical context.
- Public skill text distilled from private working history, not carelessly quoted from it.

The roadmap is to build a values map, map Superpowers skills to Hearthands equivalents, write a repeatable rewrite guide, and then replace the upstream skill set in batches. The first batch should focus on verification, debugging, and planning, because those skills shape agent honesty, risk, and handoff quality across nearly every development workflow.
